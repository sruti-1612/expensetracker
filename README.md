# Expense Tracker

## Project Description
This is a simple Python-based Expense Tracker application.

## Features
- Add expenses
- View all expenses
- Calculate total spending
- Menu-driven interface

## Concepts Used
- Variables
- Lists
- Loops
- Conditional Statements
- User Input

## How to Run

1. Download the project.
2. Open terminal/command prompt.
3. Run:

python expense_tracker.py
